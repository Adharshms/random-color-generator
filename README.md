# Random Color Generator

This is a simple Random Color Generator built using JavaScript. It allows users to generate random colors with the click of a button, providing a fun and interactive way to explore color combinations.

## Features

- Generate random colors with the click of a button.
- Displays the color in both HEX and RGB formats.
- User-friendly interface.
- Simple and lightweight.

## Installation

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Adharshms/random-color-generator.git
   ```

2. Navigate to the project directory:
   ```bash
   cd random-color-generator
   ```

3. Open the `index.html` file in your preferred web browser to start generating random colors.

   - For live preview, you can also use any code editor (like VS Code) with Live Server extension.

## Technologies Used

- **HTML**: Used to structure the layout of the page.
- **CSS**: Used for styling the page and the color display area.
- **JavaScript**: Used to generate random colors and handle the button click event to display them.

## How It Works

1. The user clicks the **Generate Color** button.
2. A random color is generated.
3. The color is displayed on the screen in both HEX and RGB format.
4. The user can continue to generate new random colors.
